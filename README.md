elasticsearch-tools
===================

A variety of tools for making managing ElasticSearch a bit easier.

Requirements
------------

Python 2.7, ElasticSearch .90+

#####Python Packages

- `elasticsearch` - Low-level ElasticSearch library.
- `argparser` - Argument parsing library.
